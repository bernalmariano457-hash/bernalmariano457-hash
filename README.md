## 🐾 Sentinel Anubis OS

Una herramienta táctica de seguridad pensada para correr en terminal, sin GUI, sin mouse. Construida sobre Python para la **ClockworkPi uConsole** — una consola con teclado físico que cabe en una mochila.

No es un wrapper de Kali ni un script de automatización. Es un sistema completo con su propio ciclo de autenticación, interfaz Rich, y módulos que se cargan según el hardware disponible.

**Lo que hace:**
- Análisis de espectro RF en tiempo real con RTL-SDR
- Decodificación ADS-B (aeronaves) y NOAA APT (satélites meteorológicos)
- Escaneo y triangulación Wi-Fi + Bluetooth BLE
- Geolocalización y mapas interactivos
- Módulos de forense, OSINT y auditoría
- Cifrado Fernet, autenticación bcrypt y protocolo de pánico

En colaboración con **ClockworkPi** para pruebas en hardware real (uConsole CM4).
